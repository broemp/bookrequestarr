# Screenshots

This directory contains screenshots of the Bookrequestarr application for the main README.

## Captured Screenshots

The following screenshots have been captured using the browser MCP tool:

1. **dashboard.png** - Main dashboard showing trending books and request statistics
2. **search.png** - Search page in empty state
3. **search-results.png** - Search results for "project hail mary"
4. **book-details.png** - Book details modal showing full information
5. **my-requests.png** - User's book requests list
6. **admin-requests.png** - Admin panel for managing all requests
7. **settings.png** - User settings page

## How to Update Screenshots

To update these screenshots:

1. Ensure the development server is running: `npm run dev`
2. Use a browser automation tool or manually capture screenshots at 1920x1080 resolution
3. Navigate to each page and capture:
   - `/dashboard` - Dashboard page
   - `/search` - Search page (empty)
   - `/search` - Search page with "project hail mary" query
   - `/search` - Book details modal (click on a book)
   - `/requests` - My Requests page
   - `/admin/requests` - Admin Requests page
   - `/settings` - Settings page

## Notes

- Screenshots should be taken at 1920x1080 resolution for consistency
- Use the dark theme (default)
- Ensure there is sample data in the database for better screenshots
- The screenshots were captured with the development admin user
